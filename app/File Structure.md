
# General Info
- "res://" refers to the root of the godot/client app (mindpalace/app/)
	- this is a actual term in godot dev
- "backend://" refers to root of the flask/server app (mindpalace/server) 
	- made up abbreviation for this project in specific for brevity

# Godot App

### res://Areas
- This directory contains scenes that are used **as** in-game areas
	- Rooms

### res://Prefabs
- This directory contains scenes that are used **inside** in-game areas
	- Decor
	- Items
	- Actors
### res://Interfaces
- This directory contains user interface scenes
	- Popups
	- Menus
	- Buttons
